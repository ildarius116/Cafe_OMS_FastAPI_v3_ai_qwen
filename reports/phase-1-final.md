- Proxy для API (`/api` → `http://localhost:8000`)
- Google Fonts (Inter, Plus Jakarta Sans)

### ✅ 1.5. Создание агентов

Создано 4 специализированных агента в папке `.qwen/agents/`:

#### `ui-designer` (cyan)
- **Назначение:** Создание UI компонентов и страниц
- **Специализация:** React + Tailwind CSS
- **Принципы:** Доступность, адаптивность, уникальная эстетика
- **Инструменты:** Read, Write, Grep, Glob

#### `code-generator` (green)
- **Назначение:** Генерация backend кода
- **Специализация:** FastAPI, SQLAlchemy, Pydantic
- **Принципы:** Типизация, чистота кода, best practices
- **Инструменты:** Read, Write, Grep, Glob

#### `code-reviewer` (yellow)
- **Назначение:** Проверка кода на ошибки и уязвимости
- **Специализация:** Security audit, PEP 8, performance
- **Чеклист:** Типизация, ошибки, безопасность, производительность
- **Инструменты:** Read, Write, Grep, Glob, Bash

#### `test-generator` (magenta)
- **Назначение:** Создание тестов
- **Специализация:** pytest, FastAPI TestClient, React Testing Library
- **Покрытие:** Unit, Integration, API, Component тесты
- **Инструменты:** Read, Write, Grep, Glob, Bash

## Дизайн-сессия с Design Lab

**Примечание:** Дизайн-сессия через `/design:start` будет проведена после установки всех зависимостей и запуска dev-сервера. В данный момент проект готов к проведению сессии.

**План сессий:**
1. Dashboard — главная страница со статистикой
2. OrdersPage — таблица заказов с фильтрами
3. UsersPage — таблица пользователей
4. LoginPage — форма авторизации

## Следующие шаги

1. **Установка зависимостей** — `npm install` в папке frontend
2. **Запуск dev-сервера** — `npm run dev`
3. **Проведение дизайн-сессий** — `/design:start [Component]`
4. **Начало ФАЗЫ 2** — инициализация Python-проекта
